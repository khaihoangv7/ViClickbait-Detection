# ViClickbait-2025 Dataset

ViClickbait-2025 is a Vietnamese-language dataset curated to support research on clickbait detection in online journalism. It contains 3,414 news headlines collected from 8 major Vietnamese news portals and annotated as clickbait or non-clickbait. The dataset is designed to reflect diverse journalistic styles, categories, and time periods in the Vietnamese digital media landscape.

This dataset was constructed as part of a broader research project aiming to improve the quality of information in Vietnamese digital media by enabling the development and benchmarking of machine learning models for automatic clickbait detection.

⚠ Note: This dataset is currently under review for public release and does not yet have an associated DOI. We will update this repository with citation information once the data article is formally accepted and published.

## Key Features

* 3,414 annotated news headlines in Vietnamese
* Labels: clickbait / non-clickbait
* Sources: 8 online news platforms, including Báo Mới, VnExpress, Thanh Niên, and others
* Metadata fields include category, publication time, and lead paragraph
* Cleaned and normalized text data, ready for NLP tasks

## Intended Use

ViClickbait-2025 is suitable for the following tasks:

* Clickbait classification (binary classification)
* Feature extraction and linguistic analysis
* Evaluation of Vietnamese NLP models (PhoBERT, Vistral, LLMs)
* Educational and benchmarking purposes
